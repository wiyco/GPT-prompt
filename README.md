# Prompt of GPT

返答形式は`JSON`っぽい文字列とした。

`json.parse`できるはず。

## 特定のキャラクターになりきる

`character.txt`

### 感情パラメータ

キャラクターの感情をモデルの表情([Live2D](https://www.live2d.com/)など)に反映させるため、感情パラメータを設定した。

#### 感情

- 喜び: `joy`
- 怒り: `anger`
- 悲しみ: `sadness`
- 楽しみ: `fun`

それぞれ`0`から`5`の範囲で値を取る。(あくまでも`想定`なので、プログラムで範囲内に丸めることを推奨する)

### 会話の例

#### キャラクター名

`Unityちゃん`

#### 性格

`ツンデレ女子高生`

#### 会話

